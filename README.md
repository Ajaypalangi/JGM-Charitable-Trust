# JGM Charitable Trust

Welcome to the official repository for the **JGM Charitable Trust** website! This is a static, responsive web application built to represent the trust and its various community service programs online.

## 🌟 About The Trust
The JGM Charitable Trust is a non-profit organization dedicated to empowering communities and providing essential services to those in need. The website serves as a central hub for information regarding the trust's programs, credentials, and contact details.

### 🎯 Key Programs & Initiatives
- **Adult & Children's Education:** Providing learning opportunities for all ages.
- **Disaster Relief & Medical Camps:** Offering critical support and healthcare in times of need.
- **Women Empowerment:** Programs dedicated to uplifting and supporting women.
- **Eco Development & Human Rights:** Advocating for the environment and basic rights.
- **Sports Activities:** Engaging communities through physical health and teamwork.

## 💻 Technical Details
This project is built as a static site focusing on performance, clean aesthetics, and responsiveness across all devices (mobile, tablet, desktop).

- **Core Technologies:** HTML5, CSS3, Vanilla JavaScript
- **Features:**
  - Modern, responsive landing page.
  - Interactive image gallery featuring a dynamic "Lightbox" to view full-sized images.
  - Organized sections for Programs, About Us, Legal Credentials, and Contact Info.
  - Custom animations and transitions for an improved user experience.
